# 🍝 Pasta Fasta – Modern Restaurant Website

A premium, modern website built for Pasta Fasta, designed to deliver a strong visual identity, smooth animations, and an engaging user experience.

This project focuses on branding, motion, and layout precision, inspired by award-winning web design standards.

---

## 🚀 Live Preview

👉 [https://github.com/kukavigan/pasta-fasta-website](https://kukavigan.github.io/pasta-fasta-website/)

---

## ✨ Features

- ⚡ Fast and responsive (Vite + React)
- Modern UI with clean typography (Plus Jakarta Sans)
- GSAP-powered animations (smooth transitions & effects)
- 📱 Fully responsive design
- Custom product orbit section
- Sticky navbar with mobile-style menu
- Animated promotional popup
- Full-screen locations section
- Stylish contact form with background image
- Social-ready (Open Graph optimized for sharing)

---

## 🛠️ Tech Stack

- React + Vite
- TypeScript
- Tailwind CSS
- GSAP (Animations)
- Lucide Icons / Custom SVG
